## 📂 数据说明

`src/data/` 目录下的所有数据文件（包括各游戏的`data.json`和`types.json`）均由后端爬虫自动生成，**请勿手动修改**这些文件。如需修改数据内容，请通过 [Issues](https://github.com/Trrrrw/hoyo_video/issues) 提出请求。

## 🚀 开发指南

```bash
# 安装依赖
yarn

# 启动开发服务器
yarn dev

# 生产环境构建
yarn build
```
